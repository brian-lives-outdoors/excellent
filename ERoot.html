bind()

Searches for all elements in the document not yet bound, and binds them to controllers.

Normally, a controller creates new controlled elements within its children, and then uses ctrl.bind() method. It is only if you create a new controlled element that's not a child element that you would use this global bind method.

And if you call it while in the process of binding, the call will be delayed, which is the scenario best to be avoided.
